## 🔌 后端接口需求

### 1. 首页接口
```javascript
GET /api/trade-tension
返回: {
  code: 200,
  data: [
    { region: '美国', level: '高', score: 85, newsCount: 156, updateTime: '...', remark: '...' }
  ]
}
```

### 2. 爬虫接口
```javascript
POST /api/crawler/news
参数: { country1, country2, dateRange, count }
返回: {
  code: 200,
  data: [
    { title, source, summary, url, publishTime, crawlTime }
  ]
}
```

### 3. NLP 接口
```javascript
POST /api/nlp/analyze-url
参数: { url }
返回: {
  code: 200,
  data: {
    sentiment: { type, label, positive, neutral, negative },
    summary: ['point1', 'point2', ...],
    keywords: ['keyword1', 'keyword2', ...]
  }
}

POST /api/nlp/analyze-text
参数: { content }
返回: 同上
```
